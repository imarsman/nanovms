Sample GRPC passthrough

This is a work in progress. So far I have figured out how to write a protocol buffer definition and compile it to bridge code. The GRPC server starts without error but I have not done the tie-together work or the handler.

The goal here is to write a simple GRPC server that calls a function that gets a link to data for a day's XKCD cartoon. The GRPC function is to be called by a client in a handler function that recieves the request from a browser. Browsers are not able to handle the GRPC protocol so go-betweening is necessary. There are formal projects for this that I will look into but for now I will do this.
